## Personal Finance System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/28406a7aed7c401ea85926e83b71389d/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Income Bill Management
6. Expense Bill Management
7. Financial Planning Management
8. Statistical Analysis Management
9. Memo Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Bill Type Management
